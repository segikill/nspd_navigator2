# Changelog

## 1.3.1-v19.11-ortho-first

- Добавлены базовые карты НСПД из `/api/geoportal/v1/baselayers`.
- Исправлен порядок базовых карт: первым идет `Единая электронная картографическая основа (ортофото)(основной слой)`.
- Добавлена выгрузка тайлов в папку с переносимым VRT и `tiles/`.
- Добавлен `metadata.json` со служебными датами, source URL и кандидатами дат из URL источника.
- Добавлен UI прогресса выгрузки тайлов.
- Поднят максимум выбора zoom для выгрузки до `z=25`.
- Добавлена защита масштаба для растровых подложек.
- Добавлены обходы SSL/сетевых особенностей для QGIS 3.44.
