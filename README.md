# NSPD Navigator

Модуль для работы с подложками и слоями НСПД в QGIS через модуль Kolba (https://plugins.qgis.org/plugins/kolba/).

## Установка

В Kolba откройте получение Web Script и укажите ссылку:

```text
https://raw.githubusercontent.com/segikill/nspd_navigator2/main/nspd_navigator.py
```

## Базовые карты и подложки

Есть подгрузка базовой основы НСПД:

- Единая электронная картографическая основа (ортофото)(основной слой);
- Единая электронная картографическая основа (основной слой);
- ДЗЗ. Российская Федерация;
- Цифровая объектовая схема;
- Ортофотопланы 2000;
- Ортофотопланы 10000.

Для базовой основы доступно:

- добавить подложку в проект QGIS;
- выбрать охват на карте;
- выбрать масштаб/zoom скачивания;
- скачать тайлы по выбранному охвату и масштабу;
- открыть результат как единую VRT-мозаику в QGIS.

## Другие слои НСПД

Есть подгрузка тематических слоев НСПД с выгрузкой векторных данных по выбранному охвату.

Группы слоев:

- Единицы кадастрового деления;
- Административные границы;
- Земельные участки;
- Объекты капитального строительства;
- Комплексы объектов;
- Зоны с особыми условиями использования территории;
- Зонирование и планирование территории;
- Природные территории;
- Зоны социального, экономического развития;
- Территории объектов культурного наследия;
- Иные территории;
- Тепловые карты;
- Негативные процессы.

Для этих слоев доступно:

- выбрать слой в дереве модуля;
- выбрать охват на карте;
- выгрузить векторные объекты НСПД;
- добавить выгруженные объекты в проект QGIS.
