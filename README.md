# NSPD Navigator для QGIS Kolba

Модуль для работы с подложками и слоями НСПД в QGIS через Kolba.

## Установка

В Kolba откройте получение Web Script и укажите ссылку:

```text
https://raw.githubusercontent.com/segikill/nspd_navigator2/main/nspd_navigator.py
```

## Базовые карты и подложки

Модуль подгружает базовые карты НСПД:

- Единая электронная картографическая основа (ортофото)(основной слой);
- Единая электронная картографическая основа (основной слой);
- ДЗЗ. Российская Федерация;
- Цифровая объектовая схема;
- Ортофотопланы 2000;
- Ортофотопланы 10000.

Для базовых карт доступно:

- добавление подложки в проект QGIS;
- выбор охвата на карте;
- выбор масштаба/zoom выгрузки;
- скачивание тайлов по выбранному охвату;
- сохранение результата как переносимая папка `VRT + tiles/ + metadata.json`.

## Векторные данные НСПД

Модуль подгружает тематические слои НСПД и позволяет выгружать объекты по выбранному охвату.

Группы слоев:

- Единицы кадастрового деления;
- Административные границы;
- Земельные участки;
- Объекты капитального строительства;
- Комплексы объектов;
- Зоны с особыми условиями использования территории;
- Зонирование и планирование территории;
- Природные территории;
- Зоны социального, экономического развития;
- Территории объектов культурного наследия;
- Иные территории;
- Тепловые карты;
- Негативные процессы.

Для векторных слоев доступно:

- выбор слоя в дереве;
- выбор охвата на карте;
- пакетная выгрузка объектов НСПД;
- добавление выгруженных объектов в проект QGIS.

## Обновление

При обновлении файла `nspd_navigator.py` нужно повышать поле `version:` в начале скрипта. Kolba сравнивает локальную версию с версией по `original_url` и показывает обновление.
